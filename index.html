Why SSL Inspector?

Developers often rely on command-line tools like OpenSSL to inspect certificates or generate SHA256 public key hashes for SSL pinning.

SSL Inspector provides the same insights through a clean macOS interface, making certificate inspection fast and intuitive.

Typical Workflow

  1. Enter a domain name
  2. Retrieve the certificate chain
  3. Copy the SHA256 public key hash
  4. Use it for SSL pinning in your app

Full Certificate Chain

Retrieve leaf, intermediate, and root certificates from any domain instantly.

Pinning-Ready Hashes

Generate SHA256 public key hashes for secure SSL pinning workflows.

Export & Download

Export certificates in DER format and download individual certs or full chains.

Common Developer Use Cases

  • Generate SHA256 hashes for SSL pinning
  • Inspect TLS certificate chains
  • Verify certificate expiration
  • Download and analyze certificate chains
  • Debug TLS configuration issues
